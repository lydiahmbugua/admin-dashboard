# Admin Dashboard

A functional, responsive admin dashboard layout built to practice complex UI design and layout implementation. This project focuses on utilizing CSS Grid and Flexbox to create a clean, organized, and scalable dashboard interface.

## 🚀 Live Demo
[https://github.com/lydiahmbugua/admin-dashboard]

## 📋 Features
- **Sidebar Navigation:** A sticky sidebar for quick access to dashboard sections.
- **Header Section:** Search functionality, notifications, and user profile display.
- **Data Cards:** Distinct sections for displaying project metrics, announcements, and trending users.
- **Responsive Layout:** Fluid design that adapts to various screen sizes, ensuring usability on desktop and tablets.
- **Visual Hierarchy:** Implemented using CSS Grid to manage complex multi-column layouts efficiently.

## 🛠 Technologies Used
- **Frontend:** HTML5, CSS3 (Grid & Flexbox)
- **Assets:** [Google Material Icons, Google Fonts, Pinterest]
